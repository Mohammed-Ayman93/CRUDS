 Overview

CRUDS is a simple Product Management System built using HTML, CSS, and JavaScript.
The application allows users to manage products through the basic data operations known as CRUD (Create, Read, Update, Delete) with an additional Search feature.

The project focuses on practicing JavaScript DOM manipulation, data storage, and building a dynamic UI without using frameworks.

 Features

- Add new products

- Display all products in a table

- Update existing products

- Delete products

- Search products by name

- Automatic total price calculation

- Responsive user interface

* Technologies Used

=> HTML5 – Structure of the application

=> CSS3 – Styling and layout

=> JavaScript (Vanilla JS) – Logic and DOM manipulation

=> LocalStorage – Store product data in the browser

 How It Works

The user enters product details such as:

~Title

~Price

~Taxes

~Ads

~Discount

~Category

~The system automatically calculates the total price.

~Product data is saved in LocalStorage.

~Users can update, delete, or search for products easily.
